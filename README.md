# ETHBadge

A simple DApp that allows you creating an Ethereum badge for your address to share or embed in your website.

## Install dependencies

```bash
npm install
```

## Run DEV server

```bash
npm run dev
```

## Build dist

```bash
npm run build
```

## Deploy to gh-pages branch

```bash
npm run deploy
```
